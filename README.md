# Python Assignment — Part 2: Data Structures

## Theme: Restaurant Menu & Order Management System

A Restaurant Order Management System built entirely using Python's core data structures — lists, dictionaries, and nested combinations of both.

## Tasks

- **Task 1** — Explore the Menu (5 marks)
- **Task 2** — Cart Operations (8 marks)
- **Task 3** — Inventory Tracker with Deep Copy (6 marks)

## How to Run

```
python part2_order_system.py
```

No external libraries needed — only `copy` from the standard library.
